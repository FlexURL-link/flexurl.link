# 🔗 FlexURL

Welcome to **FlexURL** 🚀

This is a small web service used to create simple URL redirections.
You can generate short links that redirect to any destination URL.

---

## ⚙️ What it does

* Create custom redirect links
* Manage simple URL forwarding
* Lightweight and fast redirect system
* Useful for personal projects, tools, or sharing links
